1.3. Historique des modifications

Il s'agit de la version 8.0 du livre Linux From Scratch, datant du 25 février 2017. Si ce livre est daté de plus de six mois, une version plus récente et améliorée est probablement déjà disponible. Pour en avoir le cœur net, merci de vérifier la présence d'une nouvelle version sur l'un des miroirs via http://www.linuxfromscratch.org/mirrors.html.

Ci-dessous se trouve une liste des modifications apportées depuis la version précédente du livre.
